# Clot Shortener
A custom and fast URL shortener.

### Data
* Fast shortener.
* Accessible database (MongoDB).
* Responsive for Mobile and Desktop.
* Written in TS (backend).
* Custom URL & Multi-Domain.
* Frontend made with TailWindCSS.
* One click, copy or delete.
